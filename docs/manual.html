1. 馥華怎麼用這個 App

1.1 核心概念

馥華用頭部動作觸發音符——頭轉到 9 個位置之一，對應的音就會發出來。

位置音符備註
左上 / 上 / 右上Do, Re, Mi中音域
左 / 中 / 右Fa, 休止, So中心是休息點，不發聲
左下 / 下 / 右下La, Si, Do'（高音）高音域
為什麼中心點不發聲？ 馥華是視障 + 重度肢障，需要一個明確的「家」位置可以放鬆休息，不會誤觸到音符。

1.2 嘴部控制（八度切換）

馥華可以用嘴部動作切換音域，預設是閉嘴觸發：

  • 嘴閉著：標準音域
  • 嘴張開：高八度音域

1.3 連彈模式（預設打開）

頭部停在同一區域會自動跟著節拍重複觸發音符——不用反覆轉開再回來。

2. 硬體設定（每次使用前）

馥華坐定前先擺好攝影機。鏡頭沒對好整個追蹤都會偏。

1
將鏡頭安裝到三腳架上
2
調整三腳架高度到地面標記位置（馥華視線高度）
3
三腳架放在馥華前方約 1 公尺處
4
鏡頭 USB 連接電腦
5
打開鏡頭電源（如果有開關）
標記位置的意義：地面標記是先前校正時馥華的最佳坐姿距離。三腳架每次放回標記點就不用重做校正，所有設定（profile 1/2/3）才會跟記錄的位置對齊。
注意：如果三腳架位置變了、或馥華坐姿改變很多，9 宮格位置會跑掉。處理方式：
  • 輕微偏差 → 校正面板 → 🎯 手動設定當前為中心點
  • 整個跑掉 → 校正面板 → 🌀 重新跑圓形範圍偵測

硬體就緒後 → 進入第 3 章軟體啟動。

4. 🎤 彈奏設定（逐項說明）

打開方式：右下 🎤 彈奏設定 按鈕。

4.1 🎹 主旋律 (Melody)

① 回中重置 (單次觸發)

做什麼勾選後，頭回到中心區音就會停；不勾則音自然衰減
何時調想要「按一下就停」的精準感 → 勾起來；想要鋼琴自然延音感 → 不勾
建議馥華原始 = 不勾（延音較自然，配連彈模式效果好）

② 樂器

做什麼切換主旋律音色
選項 🎹 鋼琴（真實取樣，Salamander Yamaha C5）｜ 🎛️ 合成鋼琴（離線備援）｜ 👾 8-Bit 電子音｜ 🌬️ 長笛
何時調鋼琴音質最自然但需網路第一次載入；網速差/離線用合成鋼琴；想要變化用 8-bit 或長笛
建議預設 🎹 鋼琴，網路 OK 就維持

③ 音量

做什麼主旋律整體音量
範圍0% ～ 100%，預設 50%
何時調伴奏太大蓋住主旋律 → 主旋律調大或伴奏調小

④ 長度 (音符延音時長)

做什麼每次觸發音的衰減長度
範圍0.1 秒（極短促）～ 3.0 秒（綿長），預設 1.5 秒
何時調連彈模式下太黏糊 → 縮短到 0.5-1.0；獨彈要鋼琴感 → 拉長到 2-3
建議配連彈模式 = 1.0；獨彈 = 1.5-2.0

4.2 👄 嘴部控制 (八度切換)

⑤ 啟用嘴部八度控制

做什麼勾選後，嘴部動作切換高低八度
何時關馥華嘴部無法控制 / 不想要八度切換時關掉
建議馥華原始 = 啟用

⑥ 觸發動作

做什麼決定「哪個嘴形」切換到高八度
選項 🤐 閉嘴瞬間切換（預設）— 平常自然張口微開，閉嘴 = 高八度
😮 張嘴瞬間切換— 平常嘴閉合，張開 = 高八度
何時調馥華自然嘴形是張開的 → 用「閉嘴觸發」；自然嘴形是閉合的 → 用「張嘴觸發」

4.3 🔁 連彈行為

⑦ 連彈模式 (頭停住自動重複)

做什麼頭停在同一區域 → 跟著 BPM 自動重複觸發該音
何時開想要持續演奏、不用一直轉開又回來 → 開（馥華肢障建議開）
何時關想要每個音都精準單次觸發 → 關
建議預設 打開（已記住偏好，下次自動恢復）

⑧ 連彈速度 (BPM)

做什麼連彈模式的重複頻率，與伴奏 BPM 是同一個值
範圍40（極慢）～ 180（極快），預設 100
何時調音符之間太密 → 降到 60-80；太散 → 升到 120-140
注意這條 slider 跟「伴奏設定 → 速度 BPM」雙向同步，改任一邊另一邊跟著動

5. 🎼 伴奏設定（逐項說明）

打開方式：右下 🎼 伴奏設定 按鈕。

① 選擇曲風/進行

做什麼選擇伴奏的和弦進行 / 既有曲目
選項 -- 關閉 -- ／ ⏱ 純節拍器 (4/4) ／ ⏱ 純節拍器 (3/4) ／ 🙏 奇異恩典 (3/4, F大調) ／ 🙏 祈禱 (4/4, C大調) ／ 🐦 浜千鳥 (3/4, C大調, A-B-A') ／ 🎸 流行 C 大調 (C-G-Am-F) ／ 🎻 卡農進行 ／ 🎼 基礎 I-IV-V-I
何時調練習特定曲目 → 選曲目；自由彈奏 → 選和弦進行；只要打拍 → 選純節拍器

② 練習段落 (Practice Section)

做什麼選曲目時才出現——選擇從 A / B / A' 哪一段開始
何時調想反覆練某一段（例如 B 段副歌）→ 直接跳過去

③ 伴奏音色

做什麼和弦伴奏的音色
選項 ☁️ 柔和墊音 (Soft Pad) ／ 🎹 電鋼琴 (E.Piano，預設) ／ 🎻 合成弦樂 ／ 🎸 撥弦 (Pluck) ／ 🎸 木吉他 (Guitar)
建議電鋼琴最清楚不搶風頭；想要悠遠氛圍用 Soft Pad / 弦樂

④ 分解和弦 (Arpeggio)

做什麼勾選後和弦不齊奏，改成一個音一個音上行
何時調伴奏太厚重蓋住主旋律 → 開；想要扎實節奏感 → 關

⑤ 開啟節拍器 (Click)

做什麼加一個咔咔聲幫忙打拍子
何時調配連彈練速度感 → 開；錄音時不想要 → 關
提示節拍器旁邊有紅色燈號會跟著節奏閃

⑥ 🔊 語音報讀旋律 (無障礙)

做什麼用 TTS 報讀目前該彈哪個音（適合視障使用者跟譜）
何時調馥華練新曲不熟 → 開報讀引導；熟了 → 關
附帶開啟後下方出現「語音速度」slider（0.5x ～ 2.0x，預設 1.0x）

⑦ ▶ 開始 / 停止伴奏

做什麼手動啟動 / 停止伴奏播放
注意改曲風 / BPM / 音量都不需要重啟，即時生效

⑧ 速度 (BPM)

做什麼控制節拍器 / 自動和弦 / 連彈三件事的速度
範圍40 ～ 180 BPM，預設 100
與彈奏面板同步彈奏面板有同一條 slider，雙向連動

⑨ 伴奏音量

做什麼伴奏整體音量（不影響主旋律）
範圍0% ～ 100%，預設 30%
何時調伴奏壓過主旋律 → 降到 20-30%；伴奏聽不到 → 升到 50-70%

⑩ 🎵 旋律示範 (Melody Guide)

做什麼勾選後伴奏會同時播出主旋律（適合馥華跟著聽）
何時調新曲不熟 → 開（聽範本）；熟了想自己彈 → 關
附帶開啟後下方出現「旋律音量」slider（預設 40%）

⑪ 🩰 華爾滋伴奏 (3/4 低音-和弦-和弦)

做什麼勾選後伴奏改採 oom-pah-pah 三拍模式（低音→和弦→和弦）
何時調3/4 拍曲目（奇異恩典 / 浜千鳥）建議開；4/4 拍曲目不適合

6. ⚙️ 校正與設定（逐項說明）

打開方式：右下 ⚙️ 校正與設定 按鈕。共 6 區塊：設定檔 / 錄製校正 / 靈敏度 / 座標縮放 / 視覺輔助 / 設定檔管理。

6.1 📁 設定檔切換 + baseline 還原

① 設定檔按鈕 1 / 2 / 3

做什麼切換 profile，跟左上 pill 按鈕雙向同步
注意🔒 1 原始 只暫存本機不上雲；2 / 3 才會雲端同步

② 📥 從原始 baseline 還原

做什麼把馥華原始版 baseline 複製到當前 profile
何時用當前 profile 被改壞 → 一鍵還原
限制必須先切到 2 或 3，在 🔒 1 上會被拒絕

6.2 🎯 錄製與校正（預設收折，點 title 展開）

③ 🔴 錄製位置 / 🔵 編輯參數 模式切換

做什麼紅色模式 = 點 9 宮格按鈕錄製新位置；藍色模式 = 編輯既有位置（拖拉 / 滾輪）
建議流程先紅色逐區錄一遍 → 切藍色微調

④ 🎯 手動設定當前為中心點

做什麼把馥華當下頭部位置設為「中心 = 5 號區」
何時用馥華坐姿換了 / 中心漂移 / 9 宮格整體偏一邊

⑤ 🌀 開始圓形範圍偵測 (推薦)

做什麼讓馥華輕鬆轉一個圓圈，系統自動估範圍 → 9 宮格自動分布
操作按下按鈕 → 馥華轉一圈 3-5 秒 → 再按一次結束
建議第一次設定 / 換人 / 換坐姿都建議跑這個

⑥ 9 宮格錄製按鈕（Do / Re / Mi / Fa / 休止 / So / La / Si / Do'）

做什麼紅色模式下，馥華頭轉到該音位置 → 點按鈕 → 記錄
提示已錄製的按鈕會發亮顯示

⑦ 個別點設定面板（藍色編輯模式下選中圈圈後出現）

基音 (Base Note)改該位置對應的音名（C/D/E.../下拉選擇）
升降微調♭ ♮ ♯ 三段，可在基音上 ±1 半音
個別半徑10 ～ 120，只調該圈的感應範圍（覆蓋全域設定）
拖拉編輯模式下直接拖該圈到新位置，鬆手自動存
滾輪編輯模式下滑鼠滾輪在圈上 → 縮放半徑
⚠️ Profile 鎖定提醒：在 🔒 1 上做的任何錄製 / 拖拉 / 編輯都不會上雲，重整就消失。要保留請先切到 2 或 3。

6.3 🎚️ 反應靈敏度

⑧ 反應靈敏度 (smoothing)

做什麼頭部動作 → 螢幕游標的反應快慢。值越小反應越快但容易抖；越大越平滑但延遲
範圍0.02（極靈敏）～ 0.50（極平滑），預設 0.15
何時調 馥華有頭部 tremor / 抖動 → 調大到 0.3-0.5（平滑掉抖）
反應太遲鈍跟不上 → 調小到 0.05-0.10
馥華原始0.39（偏平滑，因為有輕微 tremor）

6.4 📐 座標縮放模式

⑨ 智能 / 手動模式切換

智能 (auto)用「圓形範圍偵測」結果自動計算最佳縮放
手動 (manual)用兩條 slider 手動設定上下/左右靈敏度
建議第一次或剛跑完圓形偵測 → 用智能；想精調 → 切手動

⑩ 🔧 左右靈敏度 (yaw scale)

做什麼放大 / 縮小馥華頭部左右轉動的對應範圍
範圍50%（縮小一半）～ 300%（放大三倍），預設 100%
何時調馥華頭左右轉幅度不大、9 宮格左/右兩排很難碰到 → 調高到 150-250%，少量動作對應較大移動範圍
反過來頭一動就衝到邊緣 / 太敏感 → 調低到 70-90%

⑪ 🔧 上下靈敏度 (pitch scale)

做什麼放大 / 縮小馥華頭部上下擺動的對應範圍
範圍50% ～ 300%，預設 100%
何時調馥華頭上下擺動幅度不大、9 宮格上/下兩排很難碰到（特別是高音 La/Si/Do' 區）→ 調高到 150-250%
馥華原始145%（上下幅度小，所以放大 1.45 倍）
「上下/左右靈敏度」白話解釋：把馥華「實際能轉動的範圍」對應到「螢幕上 9 宮格的範圍」。她實際只能轉 30 度但要她碰得到 9 宮格上下兩排 → 上下靈敏度調高，把那 30 度「放大」成螢幕上的 90 度。

6.5 🔍 視覺輔助（協助者用）

⑫ 顯示觸發圈

做什麼在攝影機畫面上顯示 9 個圈圈（馥華的觸發區域）
何時關熟練後想要乾淨畫面 → 關（馥華視障本來就看不到，這純粹給協助者）

⑬ 顯示面部導引框

做什麼畫面上顯示一個框框，幫忙確認馥華的臉是否在框內
何時關表演 / 錄製不想要框框時關

⑭ 顯示軌跡紅點 & 嘴部偵測

做什麼顯示頭部追蹤紅點 + 嘴形偵測線（debug 用）
