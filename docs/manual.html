1. 馥華怎麼用這個 App

1.1 核心概念

馥華用頭部動作觸發音符——頭轉到 9 個位置之一，對應的音就會發出來。

位置音符備註
左上 / 上 / 右上Do, Re, Mi中音域
左 / 中 / 右Fa, 休止, So中心是休息點，不發聲
左下 / 下 / 右下La, Si, Do'（高音）高音域
為什麼中心點不發聲？ 馥華是視障 + 重度肢障，需要一個明確的「家」位置可以放鬆休息，不會誤觸到音符。

1.2 嘴部控制（八度切換）

馥華可以用嘴部動作切換音域，預設是閉嘴觸發：

  • 嘴閉著：標準音域
  • 嘴張開：高八度音域

1.3 連彈模式（預設打開）

頭部停在同一區域會自動跟著節拍重複觸發音符——不用反覆轉開再回來。

  • 速度跟隨 BPM（預設 100）
  • 想關掉：彈奏設定 → 取消 🔁 連彈模式 勾選

2. 協助者的日常操作

2.1 啟動 App

1
瀏覽器打開 https://m72900024.github.io/AI-Head-Motion-Tracker/
2
按中央大按鈕 「▶ 點擊開始 (啟動音效)」
3
允許攝影機權限（瀏覽器會跳通知）
4
等左上同步燈變成 🟢 已同步

2.2 認識左上區域

元素意義
🟢 已同步雲端正常，所有設定即時備份到雲端
🟡 同步中正在寫入雲端（1-3 秒）
🔴 失敗網路或雲端問題，本地仍可用
⚫ 離線沒網路，純本地模式
🔒 1 2 3三個設定檔切換（藍色＝目前使用中）
📥從原始 baseline 還原
☁️ 套用新設定有遠端更新時出現，按下才套用（避免打斷彈奏）

2.3 三個設定檔的差別

Profile用途能否覆寫雲端同步
🔒 1 原始馥華初始 CSV 校正❌ 鎖定不可寫不上雲（避免誤觸）
2 校正 A日常使用 / 自由調整✅ 可寫✅ 雲端同步
3 校正 B備用 / 實驗✅ 可寫✅ 雲端同步
建議流程：日常用 2，要嘗試新校正時切到 3。馥華原始版永遠在 🔒 1，任何時候按 📥 都能還原。

2.4 字體放大

頂部中央有三顆 A 按鈕（小 / 中 / 大），讓視力不好的協助者看得清楚。馥華本人視障，這純粹給協助者用。

2.5 收折面板

各 section 標題（例如「🎯 錄製與校正」）旁有 ▾ 箭頭，點 title 可收折/展開，畫面才不會太擠。

  • ▾ = 已展開
  • ▸ = 已收折
  • 收折狀態瀏覽器會記住，下次開仍然收折

3. 校正調整（治療師）

校正面板路徑：右側 ⚙️ 校正與設定 按鈕打開。

3.1 設定中心點

當馥華頭部「自然放鬆」的位置不在畫面中央時，按 🎯 手動設定當前為中心點 重新對齊。

3.2 圓形範圍偵測（推薦）

讓馥華輕鬆地用頭部轉一個圓圈（不用刻意到極限），系統會自動估計她的可動範圍：

1
按 🌀 開始圓形範圍偵測
2
讓馥華自然轉一圈（3-5 秒）
3
再按一次按鈕結束偵測，9 宮格位置會自動分布

3.3 9 宮格逐區錄製

若要更精準地設定每個音的位置：

1
確認在「🔴 錄製位置」模式（左上紅色按鈕）
2
讓馥華頭部轉到該音位置（例如 Do）
3
點 9 宮格中對應的按鈕（例如 Do(1)），位置就被記錄
4
9 個都錄完才完整

3.4 微調已錄製的點

切到 🔵 編輯參數 模式後：

  • 拖拉校正圈到新位置 — 鬆手自動存檔
  • 滑鼠 滾輪 在圈圈上 — 縮放感應半徑
  • 點選某個圈，下方出現「個別設定面板」：
    • 🎵 基音（Do/Re/Mi/...）
    • ♭/♮/♯ 升降微調
    • 個別半徑（覆蓋全域設定）
注意：在 🔒 1 上做的任何變更只暫存在本機（重整頁面就消失），不會上雲。要保留請先切到 2 或 3。

4. 音色與行為設定

路徑：右側 🎤 彈奏設定 按鈕打開。

4.1 樂器

選項特性備援
🎹 piano真實鋼琴取樣（Salamander Yamaha C5）離線時自動切 synth
🎛️ synthoscillator 合成音不需網路
👾 8bit方波（懷舊風）不需網路
🪈 flute正弦波（柔和）不需網路

4.2 音量 / 時長 / 回中

設定範圍說明
音量0-100%主旋律整體音量
時長0.5-5 秒每次觸發音的延音長度
回中勾 / 不勾勾選後頭回到中心會切斷音；不勾則自然衰減

4.3 嘴部控制

  • 嘴部控制 enabled：開啟才會偵測嘴形
  • 觸發模式：
    • close（閉嘴觸發）：閉嘴 = 切高音域
    • open（張嘴觸發）：張嘴 = 切高音域

5. 連彈與伴奏

5.1 連彈模式（預設打開）

位置：彈奏面板 → 🔁 連彈行為 section

頭停在同一區域會自動跟著 BPM 重複觸發音符——馥華不用一直轉開又轉回來。

5.2 BPM 速度

BPM slider 在兩個面板都能調（雙向同步）：

  • 🎤 彈奏設定 → 連彈速度（紅色 slider）
  • 🎼 伴奏設定 → 速度 BPM（粉色 slider）

BPM 同時控制三件事：連彈節奏 / 節拍器 / 自動和弦速度。

5.3 自動和弦伴奏

位置：右側 🎼 伴奏設定 按鈕打開

樂曲調性 / 拍號備註
奇異恩典C 大調 / 3/4手寫譜 + 華爾滋模式
祈禱C 大調 / 4/4含旋律示範
浜千鳥C 大調 / 3/4A-B-A' 曲式
自動和弦多種進行卡農 / 流行 / 藍調 / 爵士

5.4 華爾滋三拍模式

勾選後低音採 1-2-3 三拍 oom-pah-pah 進行（適合 3/4 拍曲目）。

6. 靈敏度與座標縮放

6.1 反應靈敏度（smoothing）

路徑：校正面板 → 🎚️ 靈敏度

  • 0.02 = 非常靈敏（頭一動就反應，可能抖動）
  • 0.15 = 預設
  • 0.50 = 非常平滑（反應遲緩，適合頭部 tremor）

6.2 座標縮放模式

模式說明
🔵 智能（auto）系統根據範圍偵測結果自動計算 yaw/pitch 縮放
⚙️ 手動（manual）左右 / 上下靈敏度手動拉條（50-300%）

6.3 個別點半徑

每個點都有自己的「感應半徑」（觸發範圍）：

  • 全域 trigger radius slider 一次調全部
  • 滑鼠滾輪在單一圈圈上 = 只調那一個
  • 選中圈圈後個別設定面板的 slider 也可調該圈

附錄 A：故障排除

狀況原因處理
攝影機沒畫面瀏覽器權限被擋網址列左邊鎖頭 → 攝影機 → 允許 → 重整
左上 🔴 紅燈網路斷或 Firebase 連不上檢查 wifi，本地仍可用
左上 ⚫ 離線純本地模式網路恢復會自動切回 🟢
位置全部跑掉誤觸或別人覆寫切到 2/3 → 按 📥
沒聲音瀏覽器音效被擋 / Tone.Sampler 載入失敗切換樂器到 synth 試試
反應太慢 / 太靈敏smoothing 不對校正面板調 🎚️ 靈敏度
頭歪掉 9 宮格不對中心點漂移按 🎯 手動設定當前為中心點

附錄 B：進階（備份 / 多裝置同步）

B.1 CSV 匯出備份

校正面板 → 💾 設定檔管理 → 📤 匯出 CSV——下載當前 profile 為 CSV 檔。

檔名格式：head_tracker_profile_{號碼}_{日期}.csv

B.2 多裝置同步原理

所有變更（位置、樂器、音量、BPM 等）會在 1-3 秒內透過 Firebase Firestore 同步到所有開著 app 的裝置。

  • 治療師在自己筆電上調整 → 馥華家電腦自動收到
  • 馥華家電腦看到左上 ☁️ 套用新設定 按鈕
  • 協助者按下才真正套用（避免打斷彈奏中）

B.3 雲端備份結構（給技術人員）

  • head_tracker_profiles/1 — 馥華原始版（前端鎖定）
  • head_tracker_profiles/2 — 校正 A
  • head_tracker_profiles/3 — 校正 B
  • head_tracker_baselines/original — 永不被覆寫的原始副本
